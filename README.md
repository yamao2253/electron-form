# electron-form

Since: 2016/04/25

electronの雛形です。  
2016/04/25に動作確認

node: v4.4.3  
npm: 3.8.7  
electron-prebuilt: 0.37.7  

### インストール
```
$ git clone https://github.com/yamao2253/electron-form.git
$ cd electron-form
$ npm install
```

package.jsonのnameは「electron-yamjack」になってますので、プロジェクトに合わせて変更します

### 実行
```
$ npm run start
```

### 配布パッケージ作成
```
$ npm run dist
```
